# Category Images

Place your category images here. The following images are required:

## Required Images:

1. **Birthday.png** - For birthday wishlists
2. **Wedding.png** - For wedding wishlists  
3. **graduation.png** - For graduation wishlists
4. **baby shower.png** - For baby shower wishlists
5. **Christmas.png** - For Christmas wishlists

## Image Specifications:

- **Format**: PNG
- **Size**: Recommended 70x70px or larger (will be scaled)
- **Background**: Transparent or white
- **Style**: Should match your app's design theme
