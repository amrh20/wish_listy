# Bottom Sheet Decorative Vectors

This folder contains decorative character/vector images that appear at the top of bottom sheets throughout the app.

## Required Images

Place the following PNG images (70x70px recommended, transparent background):

### 1. menu_character.png
**Usage**: Edit/Share/Delete menus
**Suggestion**: Character holding a menu or list
**Example searches on Storyset.com**: "menu character", "settings cute"

### 2. friends_character.png
**Usage**: Friend selection, invitation screens
**Suggestion**: Multiple characters together or character waving
**Example searches**: "friends happy", "people group cute"

### 3. creation_character.png
**Usage**: Creating wishlists, events, adding items
**Suggestion**: Character with sparkles, gift box, or creating something
**Example searches**: "gift character", "creative happy", "magic stars"

### 4. filter_character.png
**Usage**: Sort and filter options
**Suggestion**: Character with magnifying glass or organizing items
**Example searches**: "search character", "organize cute"

### 5. settings_character.png
**Usage**: Profile settings, preferences
**Suggestion**: Character with gear/settings icon or adjusting something
**Example searches**: "settings character", "profile cute"

### 6. celebration_character.png
**Usage**: Success actions, completions
**Suggestion**: Character celebrating, confetti, party hat
**Example searches**: "celebration happy", "success character", "party cute"

## Image Specifications

- **Format**: PNG with transparent background
- **Size**: 70x70px (or larger, will be scaled)
- **Style**: Cute, colorful, friendly characters
- **Colors**: Should complement the app's purple/pink gradient theme

## Recommended Sources

1. **Storyset by Freepik** (https://storyset.com) - Best option
2. **unDraw** (https://undraw.co)
3. **Blush** (https://blush.design)
